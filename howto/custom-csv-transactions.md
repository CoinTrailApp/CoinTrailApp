Download [sample.csv](/assets/whitepaper-sample.csv) and use your favorite editor to prepare a transaction file with the same columns layout.

---------------|-------------|-------------|--------------
| xId          | Int64       |             |             |

| Asset        | String      |             |             |

| Currency     | String      |             |             |
| Total        | Number      |             |             |
| Fee          | Number      |             |             |

| Description  | String      |             |             |
